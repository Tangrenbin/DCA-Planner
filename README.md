# DCA-Planner

一个纯前端的定投收益计算器，用于快速估算不同资产配置在长期定投下的本金、收益和总金额变化。

## 在线访问
`https://dca.990124.xyz/`

## 功能概览
- 输入“每个交易日定投总金额（元）”，自动计算多种组合方案。
- 内置 5 种配置：稳健型、均衡型（推荐）、进取型、纯标普500型、纯纳斯达克型。
- 输出 5/10/15/20/25/30 年的本金、收益、总金额（单位：万元）。
- 支持深色/浅色主题切换。
- 支持将结果导出为 `.txt` 文件。

## 快速开始
项目无需安装依赖，直接用静态文件运行即可。

```bash
# 在仓库根目录启动本地静态服务
python3 -m http.server 8000
```

`http://localhost:8000/`

## 计算模型与参数
- 年交易日：`252`
- 资产年化收益率（预设）：
  - 纳斯达克100：`12%`
  - 标普500：`10%`
  - 黄金：`6%`
  - 债券：`3.5%`
- 通过年化收益率换算交易日收益率后按日复利累计。
- 当日收益率接近 0 时，按线性累计本金处理，避免除零误差。

## 使用流程
1. 输入每日定投金额（需大于 0）。
2. 点击“计算收益”查看各方案对比。
3. 如需留档，点击“下载 TXT”导出结果。

## 注意事项
- 本工具用于收益测算演示，不构成任何投资建议。
- 预期收益率为固定假设值，实际市场表现可能明显偏离。
- 当前仓库未配置自动化测试，修改后请至少手动验证：输入校验、结果渲染、TXT 导出、主题切换和移动端布局。
